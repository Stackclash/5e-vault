# Mood Cap
*Wondrous item, common*  

- **Properties**: [Crafted by](/compendium/rules/item-properties.md#Crafted%20by)

This cap is usually translucent, but changes colour to match the mood of its wearer, as described below:

**Mood Cap Colours**

| Emotion | Colour |
|---------|--------|
| Anger | Red |
| Disgust | Green |
| Fear | Purple |
