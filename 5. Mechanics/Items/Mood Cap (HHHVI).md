# Mood Cap
*Wondrous item, common*  

- **Properties**: [[/5. Mechanics/Rules/Item Properties.md#Crafted%20by\|Crafted by]]

This cap is usually translucent, but changes colour to match the mood of its wearer, as described below:

**Mood Cap Colours**

| Emotion | Colour |
|---------|--------|
| Anger | Red |
| Disgust | Green |
| Fear | Purple |
| Happiness | Yellow |
| Sadness | Blue |
