# Elemental Gem, Emerald
*Wondrous item, minor, uncommon*  


This gem contains a mote of elemental energy. When you use an action to break the gem, a [[5. Mechanics/Bestiary/Elemental/Water Element(AL).md\|water elemental]] is summoned as if you had cast the [[5. Mechanics/Spells/Conjure Element(AL).md\|conjure elemental]] spell, and the gem's magic is lost.

*Source: Dungeon Master's Guide p. 167. Available in the <span title='Systems Reference Document (5.1)'>SRD</span>*
