# Stars and Birds Perceive All
*Source: Deep Magic p. 220*  

This symbol represents vision, sight, and perceiving from great distances.

**Benefit:** You gain a +1 bonus on Wisdom ([[/5. Mechanics/Rules/Skills.md#Perception\|Perception]]) checks.

**Power:** If you use an action to trace this glyph on a creature, you can cast [[5. Mechanics/Spells/Enhance Ability.md\|enhance ability]] on the target, which gains the benefit of the Owl's Wisdom effect.

**Mastery Power:** By tracing this glyph over and over in a 10-minute ritual, you can cast Clairvoyance.
